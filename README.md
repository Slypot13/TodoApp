# 📝 Todo App - Projet Node.js

## 📌 Description

Application Todo List réalisée avec Node.js et Express.

Elle permet de :
- afficher les tâches
- ajouter une tâche
- modifier une tâche
- supprimer une tâche

Le projet contient :
- un backend (API)
- un frontend (interface)

---

## 🧱 Structure


backend/ → API (Express, routes, data.json)
frontend/ → HTML, CSS, JavaScript


---

## 🚀 Lancer le projet

Dans le dossier backend :


npm install
node app.js


Serveur disponible sur :


http://localhost:3000
http://localhost:3000/tasks


---

## 🌐 Utilisation

Ouvrir :


frontend/index.html


Fonctionnalités :
- ajouter une tâche
- voir la liste
- modifier (terminé / non terminé)
- supprimer

---

## 🔌 API

- GET /tasks
- POST /tasks
- PUT /tasks/:id
- DELETE /tasks/:id

---

## ⚠️ Gestion des erreurs

- Backend : codes 400, 404, 500
- Frontend : affichage des erreurs
